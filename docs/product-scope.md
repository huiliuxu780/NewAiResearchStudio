# AI 研究平台 - 产品范围定义

## 1. 产品定位

### 1.1 产品本质

AI 研究平台是一个面向 AI 研究工作的情报中台，不是普通的资讯聚合站或新闻阅读器。

**核心特征：**
- 面向特定研究对象的定向监控系统
- 具备"原始内容 → 结构化事实 → 研究结论"分层能力
- 能沉淀研究资产、支持复盘和持续追踪

### 1.2 产品不是什么

- ❌ 不是单纯 RSS 阅读器
- ❌ 不是普通新闻聚合网站
- ❌ 不是单纯爬虫后台
- ❌ 不是一开始就做全能 agent 平台

### 1.3 产品是什么

- ✅ 面向 AI 研究工作的情报中台
- ✅ 面向特定研究对象的定向监控系统
- ✅ 具备分层处理能力的研究平台
- ✅ 能沉淀研究资产、支持复盘和持续追踪的平台

## 3. 核心链路

### 3.1 信息处理流水线

```
信息源 → 原始记录 → 标准化事实 → 研究结论 → 日报/周报/专题研究输出
```

### 3.2 各环节说明

| 环节 | 输入 | 输出 | 核心能力 |
|------|------|------|----------|
| 信息源 | 配置的来源 URL | 采集任务 | 来源管理、调度配置 |
| 原始记录 | 网页内容 | 结构化原始数据 | 抓取、去重、存储 |
| 标准化事实 | 原始记录 | 结构化事实 | AI 抽取、分类、置信度评估 |
| 研究结论 | 标准化事实 | 洞察与建议 | AI 分析、趋势判断 |
| 报告输出 | 研究结论 | 日报/周报/专题 | 自动生成、人工复核 |

## 6. 成功指标

### 6.1 Phase 1 验收标准

- [ ] 有完整的信息架构
- [ ] 有统一数据契约
- [ ] 有完整中文字段展示
- [ ] 有 Dashboard / 信息源管理页 / 原始记录页 / 标准化事实页 / 研究结论页
- [ ] 页面风格统一、可扩展
- [ ] 可继续接入真数据

### 6.2 整体成功指标

- 信息采集覆盖率：目标公司核心信息源 100% 覆盖
- 事实抽取准确率：结构化事实准确率 > 85%
- 研究结论价值：每周产出有效洞察 > 5 条
- 研究效率提升：情报整理时间减少 50%
